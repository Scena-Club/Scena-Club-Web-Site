# Scena Web Site

This is the Curent Web Site of the Scientific Club of Electronics, Networking and Automation.

Currently it is a Vite + React + Tailwind CSS project, but this could evolve at any Time

## Improving the Web Site

If you are interested in improving the Web Site, you can do so by following these steps:

1. Fork the repository
2. Clone your forked repository
3. Create a new branch for your changes
4. Make your changes

Any Suggestion is Welcome.
